# Электронный протокол для футбольных судей

React приложение для ведения электронного протокола футбольного матча.

## Функциональность

- Ввод данных матча (турнир, дата, стадион, команды, судьи)
- Управление составами команд
- Добавление/удаление событий матча:
  - Голы (включая автоголы и пенальти)
  - Желтые/красные карточки
  - Замены
  - Травмы
  - Штрафные, угловые, офсайды и другие события
- Статистика в реальном времени:
  - Счет
  - Карточки по командам
  - Бомбардиры
- Экспорт данных в JSON формате

## Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Просмотр продакшен сборки
npm run preview
```

## Структура проекта

```
src/
  components/
    ProtocolForm.tsx    # Форма ввода данных матча
    EventsTable.tsx     # Таблица событий и форма добавления
    Statistics.tsx      # Статистика матча
  types/
    index.ts           # TypeScript типы
  App.tsx              # Главный компонент
  main.tsx             # Точка входа
  index.css            # Стили
```

## Использование

1. Заполните форму "Данные матча" (турнир, дата, стадион, команды, судьи)
2. Введите составы команд (по одному игроку на строку в формате: `номер. ФИО`)
3. Нажмите "Сохранить данные"
4. Добавляйте события матча через кнопку "Добавить событие"
5. Просматривайте статистику в реальном времени
6. Экспортируйте данные в JSON для сохранения
